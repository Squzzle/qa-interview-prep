Данный проект является приложением для подготовки к собеседованиям по тестированию. Проект представляет из себя сайт на котором будут находится стандартные вопросы с собдеседований за 2025-2026 год. Список структурированных вопросов можно увидеть в статье https://habr.com/ru/articles/1007736/.

Стек:
 - TypeScript
 - Docusaurus
 - React

Проект бьется на две части:

1. Тебе нужно создать ресурс с вопросами с такой же структурой как в статье. Один вопрос одна страница в проекте. Так же тебе необходимо написать структурированный ответ на каждый вопрос такой, что бы он удовлетворил любого интервьюера

2. После выполнения первой части нужно сделать блок для проверки усвоения материала: создать базу вопросов так что бы на каждый вопрос был перечень из 5 ответов и при этом правельных ответов могло быть от одного до пяти. Далее у пользователя есть кнопка "Пройти тест" - при нажатии на нее пользователю предлгается пройти тест из 20 вопросов из нашей базы. Если пользователь дает неверный/неполный ответ вывести ему ссылку на то где в нашей системе разбирается эта тема

Важно!! В итоге мы так же должны иметь Dockerfile для сброки image и Helm-chart для поднятия проект в k8s (на текущей машине есть minikube, можешь в нем создать namespace interview-question для нужд данного проекта)
