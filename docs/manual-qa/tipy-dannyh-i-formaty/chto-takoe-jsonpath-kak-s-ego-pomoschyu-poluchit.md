# Что такое **JSONPath**? Как с его помощью получить значение поля?

**Коротко:** JSONPath — язык запросов для навигации и извлечения данных из JSON-документа, по аналогии с XPath для XML.

## Развёрнутый ответ

JSONPath позволяет обращаться к элементам JSON-структуры по пути, не разбирая документ вручную. Он широко используется в автотестах API (например, в Rest Assured, Postman, Karate) для извлечения значений из тела ответа и последующих проверок или передачи в другой запрос.

Основной синтаксис JSONPath:

- `$` — корень документа.
- `.` или `[]` — доступ к дочернему элементу (`$.name` или `$["name"]`).
- `[n]` — доступ к элементу массива по индексу (`$.items[0]`).
- `[*]` — все элементы массива или объекта (wildcard).
- `..` — рекурсивный поиск на любом уровне вложенности (`$..price`).
- `[start:end]` — срез массива.
- `[?(@.field == value)]` — фильтр по условию.
- `@` — текущий обрабатываемый элемент (используется внутри фильтров).

JSONPath отличается от JSON Schema тем, что JSON Schema описывает структуру и валидирует документ, а JSONPath извлекает конкретные значения из уже существующего документа.

## Пример / когда применяется

Дан JSON:

```json
{
  "store": {
    "books": [
      { "title": "Клин Кода", "price": 25 },
      { "title": "Рефакторинг", "price": 30 }
    ]
  }
}
```

Примеры выражений JSONPath:

- `$.store.books[0].title` → `"Клин Кода"` — название первой книги.
- `$.store.books[*].price` → `[25, 30]` — все цены книг.
- `$..price` → `[25, 30]` — рекурсивный поиск всех полей price.
- `$.store.books[?(@.price > 25)].title` → `["Рефакторинг"]` — книги дороже 25.

Пример использования в Rest Assured (Java):

```java
String title = given()
    .when().get("/store")
    .then().extract().path("store.books[0].title");
```

## На что смотрит интервьюер

Интервьюер проверяет, умеет ли кандидат писать реальные JSONPath-выражения, а не только знает термин. Частые follow-up вопросы: как извлечь значение из массива по условию, как найти поле на любом уровне вложенности (`..`), в чём разница между JSONPath и XPath. Красный флаг — путаница между JSONPath и JSON Schema или неспособность написать простой путь к вложенному полю.
