# Таблица Employees с колонками name, salary. Вывести сотрудников с зарплатой от 3000 до 7000.

**Коротко:** диапазонная фильтрация через `WHERE salary BETWEEN 3000 AND 7000`, включающая обе границы, либо эквивалент через `>=` и `<=`.

## Развёрнутый ответ

Формулировка "от 3000 до 7000" в большинстве интерпретаций подразумевает включительный диапазон — то есть значения 3000 и 7000 входят в результат. Для этого в SQL есть специальный оператор `BETWEEN ... AND ...`, который является синтаксическим сахаром и эквивалентен условию `salary >= 3000 AND salary <= 7000`.

Ключевые нюансы:

- `BETWEEN` всегда включает обе границы (closed interval). Если нужен полуоткрытый интервал (например, исключая верхнюю границу), `BETWEEN` не подходит — нужно писать явное условие с `<`.
- Порядок аргументов важен: `BETWEEN 3000 AND 7000` работает, а `BETWEEN 7000 AND 3000` в стандартном SQL вернёт пустой результат, так как трактуется как `salary >= 7000 AND salary <= 3000`.
- Для дат `BETWEEN` также работает, но нужно быть внимательным с временной частью (`TIMESTAMP`), так как верхняя граница может "отрезать" события, произошедшие в последний день после полуночи.
- С точки зрения производительности `BETWEEN` на индексированной колонке позволяет оптимизатору использовать range scan по индексу.

## Пример / когда применяется

Такой запрос типичен для сегментации сотрудников по зарплатным вилкам, например для анализа распределения окладов внутри диапазона.

```sql
SELECT name, salary
FROM Employees
WHERE salary BETWEEN 3000 AND 7000;
```

Эквивалентный вариант:

```sql
SELECT name, salary
FROM Employees
WHERE salary >= 3000 AND salary <= 7000;
```

## На что смотрит интервьюер

- Знает ли кандидат, что `BETWEEN` включает обе границы, и может ли объяснить это словами "включительно".
- Умеет ли переписать `BETWEEN` через `>=`/`<=` — базовая проверка понимания синтаксического сахара.
- Обращает ли внимание на порядок границ в `BETWEEN`.
- Follow-up: "а как исключить нижнюю границу?" — ожидается замена на `salary > 3000 AND salary <= 7000`.
