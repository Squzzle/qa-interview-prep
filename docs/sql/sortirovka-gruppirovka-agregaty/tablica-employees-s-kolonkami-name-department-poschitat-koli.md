# Таблица Employees с колонками name, department. Посчитать количество сотрудников в каждом отделе.

**Коротко:** нужно сгруппировать строки по колонке department через `GROUP BY department` и применить `COUNT(*)` к каждой группе.

## Развёрнутый ответ

`GROUP BY` разбивает все строки таблицы на группы по значениям одной или нескольких колонок. К каждой такой группе агрегатные функции (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`) применяются отдельно, и в результате получается по одной строке на группу.

Ключевые правила:

- В `SELECT` при использовании `GROUP BY` можно указывать только те колонки, которые либо перечислены в `GROUP BY`, либо являются результатом агрегатной функции. Иначе будет ошибка (в строгих СУБД, например PostgreSQL) или неопределённое поведение (в MySQL со старыми настройками).
- Если в колонке `department` есть `NULL`, то все строки с `NULL` объединяются в отдельную группу — `NULL` считается одним значением группировки.
- Порядок групп в результате не гарантирован, если явно не задан `ORDER BY`.
- Для подсчёта количества сотрудников в группе используется `COUNT(*)` (считает все строки группы) — это не то же самое, что `COUNT(name)`, который проигнорирует строки с `NULL` в имени.

## Пример / когда применяется

Используется для отчётов по штатной численности отделов.

```sql
SELECT department, COUNT(*) AS employee_count
FROM Employees
GROUP BY department
ORDER BY employee_count DESC;
```

## На что смотрит интервьюер

- Забывают добавить `department` в `SELECT` или ставят в `SELECT` неагрегированную колонку, не входящую в `GROUP BY` (например, `name`) — это ошибка.
- Путают фильтрацию до и после группировки: `WHERE` фильтрует строки до группировки, `HAVING` — группы после группировки.
- Follow-up: "Как отсортировать отделы по количеству сотрудников?" — проверка комбинации `GROUP BY` + `ORDER BY` по алиасу агрегата.
- Follow-up: "Что если department может быть NULL?" — проверка понимания группировки NULL-значений.
- Красный флаг: попытка использовать `DISTINCT` вместо `GROUP BY` там, где нужен именно подсчёт по группам.
