# QA Interview Prep Site — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Собрать статический сайт-тренажёр для QA-собеседований (каталог вопросов + интерактивный тест) и упаковать его в Docker-образ и Helm-chart для деплоя в minikube.

**Architecture:** Docusaurus (docs-mode, TypeScript, RU) генерирует статику; контент — Markdown-страницы (1 вопрос = 1 страница) с автогенерируемым сайдбаром; тест — React-компонент с чистой функцией оценки, читающий JSON-базу вопросов, всё считается на клиенте. Статика отдаётся nginx в multi-stage Docker-образе; Helm-chart разворачивает stateless Deployment + ClusterIP Service в namespace `interview-question`.

**Tech Stack:** TypeScript, Docusaurus 3 (classic preset), React, Vitest (юнит-тесты чистой логики), Docker (node:22-alpine → nginx:alpine), Helm 4, minikube.

## Global Constraints

- Node.js 22, npm (lockfile `package-lock.json`).
- Docusaurus classic preset, TypeScript, дефолтная локаль `ru`.
- Docs отдаются по префиксу `/docs` (routeBasePath по умолчанию). Все `topicUrl` в тестовой базе имеют вид `/docs/<section>/<block>/<slug>`.
- `onBrokenLinks: 'throw'` — битые ссылки ломают сборку.
- Автогенерируемый сайдбар: `{type: 'autogenerated', dirName: '.'}`; порядок/подписи блоков — через `_category_.json`.
- Хранилище: контент — `docs/**/*.md`; тестовая база — `data/quiz/*.json` (каждый файл — массив `QuizQuestion`), грузится через `require.context('@site/data/quiz', ...)`.
- Docker-образ: `interview-question:local`, `imagePullPolicy: IfNotPresent`, собирается в docker-демоне minikube.
- Kubernetes namespace: `interview-question`. Service тип `ClusterIP`, порт `80`. Доступ — `kubectl port-forward`.
- Объём контента: Manual QA (18 блоков), Automation QA (12 блоков, концептуально), Лайвкодинг SQL (~30 задач), HR/логика. Исключены «Вопросы по Java» и «Лайвкодинг Java».
- Шаблон страницы-ответа (§ шаблон ниже) обязателен для каждого вопроса.
- Все коммиты завершаются строкой `Co-Authored-By: Claude Opus 4.8 (1M context) <noreply@anthropic.com>`.

```ts
export function isCorrect(selected: number[], correct: number[]): boolean {
  const a = [...new Set(selected)].sort((x, y) => x - y);
  const b = [...new Set(correct)].sort((x, y) => x - y);
  return a.length === b.length && a.every((v, i) => v === b[i]);
}
```

```ts
export function selectRandom<T>(items: T[], n: number, rng: () => number = Math.random): T[] {
  const pool = [...items];
  const count = Math.min(n, pool.length);
  const result: T[] = [];
  for (let i = 0; i < count; i++) {
    const idx = Math.floor(rng() * pool.length);
    result.push(pool.splice(idx, 1)[0]);
  }
  return result;
}
```

- [ ] **Step 1: Компонент `Quiz.tsx`**

```tsx
import React, {useMemo, useState} from 'react';
import Link from '@docusaurus/Link';
import {loadQuizBank} from './bank';
import {selectRandom} from './select';
import {isCorrect} from './grading';
import type {QuizQuestion} from './types';
import styles from './Quiz.module.css';

const TEST_SIZE = 20;

export default function Quiz(): React.ReactElement {
  const bank = useMemo(() => loadQuizBank(), []);
  const [round, setRound] = useState(0);
  const [started, setStarted] = useState(false);
  const [answers, setAnswers] = useState<Record<string, number[]>>({});
  const [submitted, setSubmitted] = useState(false);

  const questions = useMemo<QuizQuestion[]>(
    () => (started ? selectRandom(bank, TEST_SIZE) : []),
    [started, round, bank],
  );

  function toggle(qid: string, idx: number) {
    setAnswers((prev) => {
      const cur = prev[qid] ?? [];
      return {...prev, [qid]: cur.includes(idx) ? cur.filter((i) => i !== idx) : [...cur, idx]};
    });
  }

  function start() {
    setAnswers({}); setSubmitted(false); setStarted(true); setRound((r) => r + 1);
  }

  if (!started) {
    return (
      <div className={styles.center}>
        <p>В базе {bank.length} вопросов. Тест — {Math.min(TEST_SIZE, bank.length)} случайных.</p>
        <button className="button button--primary button--lg" onClick={start}>Пройти тест</button>
      </div>
    );
  }

  if (submitted) {
    const wrong = questions.filter((q) => !isCorrect(answers[q.id] ?? [], q.correct));
    const score = questions.length - wrong.length;
    return (
      <div>
        <h2>Результат: {score} / {questions.length}</h2>
        {wrong.length === 0 ? (
          <p>Все ответы верны! 🎉</p>
        ) : (
          <>
            <p>Разберите темы, где были ошибки:</p>
            <ul>
              {wrong.map((q) => (
                <li key={q.id}>
                  {q.question} — <Link to={q.topicUrl}>Разобрать тему</Link>
                </li>
              ))}
            </ul>
          </>
        )}
        <button className="button button--primary" onClick={start}>Пройти заново</button>
      </div>
    );
  }

  return (
    <div>
      {questions.map((q, qi) => (
        <fieldset key={q.id} className={styles.q}>
          <legend><b>{qi + 1}.</b> {q.question}</legend>
          {q.options.map((opt, idx) => (
            <label key={idx} className={styles.opt}>
              <input
                type="checkbox"
                checked={(answers[q.id] ?? []).includes(idx)}
                onChange={() => toggle(q.id, idx)}
              />{' '}{opt}
            </label>
          ))}
        </fieldset>
      ))}
      <button className="button button--primary button--lg" onClick={() => setSubmitted(true)}>
        Проверить
      </button>
    </div>
  );
}
```

- [ ] **Step 1: Написать валидатор `scripts/validate-quiz.mjs`**

```js
import {readFileSync, readdirSync, existsSync} from 'node:fs';
import {join} from 'node:path';

const quizDir = 'data/quiz';
const files = readdirSync(quizDir).filter((f) => f.endsWith('.json'));
let count = 0, errors = 0;

// URL /docs/a/b/slug -> файл docs/a/b/slug.md (или .mdx)
function topicExists(url) {
  const rel = url.replace(/^\/docs\//, 'docs/').replace(/\/$/, '');
  return existsSync(rel + '.md') || existsSync(rel + '.mdx') || existsSync(join(rel, 'index.md'));
}

for (const f of files) {
  const items = JSON.parse(readFileSync(join(quizDir, f), 'utf8'));
  for (const q of items) {
    count++;
    const bad = [];
    if (!q.id) bad.push('id');
    if (!Array.isArray(q.options) || q.options.length !== 5) bad.push('options!=5');
    if (!Array.isArray(q.correct) || q.correct.length < 1 || q.correct.length > 5) bad.push('correct len');
    if ((q.correct || []).some((i) => i < 0 || i > 4)) bad.push('correct index');
    if (!q.topicUrl || !topicExists(q.topicUrl)) bad.push('topicUrl missing: ' + q.topicUrl);
    if (bad.length) { errors++; console.error(f, q.id, bad.join(', ')); }
  }
}
console.log(`Проверено вопросов: ${count}, ошибок: ${errors}`);
process.exit(errors ? 1 : 0);
```
